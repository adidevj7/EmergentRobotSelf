# Code Availability

Code for analysis and experiments associated with:

**Evidence of an Emergent Self in Multibehavior Robot Training**  
Adidev Jhunjhunwala, Judah Goldfeder, Hod Lipson

Repository: https://github.com/adidevj7/EmergentRobotSelf

This repository contains:
- Analysis scripts/notebooks
- Training/collection utilities
- Essential lightweight model checkpoints (`.pth`) required to reproduce the reported analyses

For an archival, citeable snapshot corresponding to a specific manuscript version, create a GitHub Release
and optionally mirror it to an external archive (e.g., Zenodo) for a DOI.
